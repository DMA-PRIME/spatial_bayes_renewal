FEATURES
========

TWO MODELS IN ONE PACKAGE:

1. ClassicalForecaster
   - Non-spatial model for single or aggregated regions
   - Supports multiple infection dynamics: Basic, Feedback, Logistic_S
   - Hospital admissions + wastewater observations
   - Fast inference for single-region scenarios
   
2. SpatialForecaster  
   - Multi-region model with spatial spillover effects
   - Network-based adjacency matrices for region connectivity
   - Vectorized JAX operations for efficiency
   - Region-level heterogeneity with shared hyperparameters


QUICK USAGE
===========

# Classical (single region)
from spatial_bayes_renewal import ClassicalForecaster

forecaster = ClassicalForecaster(
    df_data_train=data,
    cols_concern=['hosp_obs', 'ww_obs'],
    n_forecast_points=14,
    pop=1000000,
    num_samples=200
)
samples, predictions = forecaster.run_mcmc()


# Spatial (multiple regions)
from spatial_bayes_renewal import SpatialForecaster
import networkx as nx

G = nx.DiGraph()
G.add_edge('region1', 'region2', weight=0.5)

forecaster = SpatialForecaster(
    df_data=data,
    spatial_net=G,
    region_list=['region1', 'region2'],
    region_obs_list=['region1', 'region2'],
    cols_concern=['hosp_obs', 'ww_obs'],
    n_forecast_points=14
)
samples, predictions = forecaster.run_mcmc()

NEXT STEPS
===========

1. Install the package:
   cd spatial_bayes_renewal
   pip install -e .

2. Choose your model:
   - Use ClassicalForecaster for single-region studies
   - Use SpatialForecaster for multi-region with spatial networks

3. Prepare your data:
   - Set up distribution arrays: gen_int_array, inf_hosp_array, sheddin
   - Provide population(s)
   - Format observations (hosp_obs, ww_obs)

4. Run inference:
   samples, predictions = forecaster.run_mcmc()

5. Analyze results:
   - Use posterior samples for credible intervals
   - Plot predictions with uncertainties
   - Compare models if applicable
